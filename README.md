# Applied-Machine-Learning-for-Food-Delivery-Operations-Clustering-Regression-and-Classification

Overview

This project implements an end-to-end machine learning workflow for analysing food delivery operations, predicting delivery time, and modelling the likelihood of customer complaints.

The project uses a historical food delivery dataset containing order-level, pricing, workload, timing, and complaint-related features. The aim was to apply appropriate machine learning techniques to real-world operational problems, including unsupervised clustering, regression, and binary classification.

The work focuses not only on model performance, but also on model selection, preprocessing design, feature engineering, evaluation strategy, baseline comparison, hyperparameter tuning, and critical interpretation of results.
